# Popup Builder - Framer Plugin

A comprehensive popup/modal builder plugin for Framer. Create, manage, and deploy popups with a visual editor, A/B testing, and analytics.

## Features

- **6 popup types**: Modal, Slide-in, Fullscreen, Banner (top/bottom), Toast
- **Campaign management**: Create and manage multiple popups
- **Visual editor**: Colors, gradients, images, border radius, shadows, custom CSS
- **Form builder**: Email, name, phone, and custom fields with drag reorder
- **6 trigger types**: Time delay, scroll %, exit intent, click selector, page load, inactivity
- **Targeting rules**: URL match, device type, visitor type, referrer, date scheduling, frequency
- **A/B testing**: Up to 5 variants with weighted traffic split
- **Integrations**: Mailchimp, ConvertKit, custom webhook
- **Analytics dashboard**: Impressions, conversions, close rate, device breakdown
- **12 templates**: Newsletter, discount, announcement, social proof, countdown, feedback, lead magnet, webinar, cookie wall, age gate, exit offer, welcome back
- **Self-contained embed code**: No external dependencies

## Development

npm install

```

```

- React 18 + TypeScript

- Framer Plugin SDK
